## Featured projects

### [GravSense](https://github.com/amgharhind/gravsense) — Debris Detection & Volume Estimation
> Open-vocabulary construction debris segmentation and volume estimation from a single photo.

- **Grounded SAM** pipeline: GroundingDINO text prompt → NMS → SAM pixel masks
- **Depth Anything V2** for automatic pile-height estimation (monocular metric depth)
- **GroundingDINO auto-calibration** for real-world scale via vehicle detection
- Production FastAPI service · Docker Compose · GitHub Actions CI/CD · GHCR

`Python` `PyTorch` `FastAPI` `Docker` `GroundingDINO` `SAM` `Depth Anything V2`

---

### [RL Graph Memory Agent](https://github.com/amgharhind/rl-graph-memory-agent) — PPO Navigation
> PPO agent trained to navigate a knowledge graph, handling indirect phrasing and multi-hop reasoning.

- 69.6% → 97.8% accuracy on 46 adversarial test cases
- Graph embeddings + policy network for when-to-stop decisions

`Python` `PPO` `Reinforcement Learning` `NetworkX`
